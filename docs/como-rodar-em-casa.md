# Como rodar em casa

## Caminho recomendado

1. Instalar `helm` e `kubectl`.
2. Subir um cluster local (ex.: `k3d`, `kind`, `minikube`) ou usar cluster remoto.
3. Executar o laboratório da sessão 1.
4. Executar o laboratório da sessão 2.

## Ordem de estudo

- `labs/sessao-1/tutorial.md`
- `labs/sessao-2/tutorial.md`

## Dica

Se não tiver cluster disponível, execute ao menos a parte de validação local com `helm lint` e `helm template`.
